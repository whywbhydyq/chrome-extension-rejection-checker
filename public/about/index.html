About this local Chrome extension checker

Chrome Extension Rejection Checker is a free browser-based preflight tool for developers preparing Manifest V3 extension ZIP files for Chrome Web Store review.

What it does

  • Reads the selected extension ZIP locally in your browser.
  • Checks package structure, manifest references, remote hosted code signals, dynamic code patterns, CSP, permissions, and privacy review reminders.
  • Shows a static report so you can fix obvious package-level issues before submission.

What it does not do

It is not an official Chrome Web Store validator, does not upload source code, does not inspect Developer Dashboard fields, and does not guarantee approval.
